### Summary

Assemble is a full-stack resource management tool for small craftsmanship enterprises. Built with Spring Boot, Next.js,
PostgreSQL, and MongoDB, all Dockerized for easy deployment. It features project, employee, and user management, work
time tracking with a calendar, and holiday imports from external sources, providing a centralized, efficient solution
for daily operations.

### Structure

- `backend/` – Spring Boot API connected to dockerized MongoDB & Postgresql.
- `frontend/` – Nextjs frontend.
